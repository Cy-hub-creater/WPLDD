NGR-for-Wheat-Disease-Identification

官方PyTorch实现 | 论文处于投刊阶段，标题：《NGR: A Novel and High Accuracy Deep Learning Model for Wheat Disease Identification》

提出一种基于改进ResNet和双注意力机制的深度学习模型，实现小麦叶片常见病害与健康叶片的高精度分类，辅助农业病害快速诊断。

1. 研究背景与模型定位

小麦作为中国第二大粮食作物，其叶片病害（如白粉病、斑枯病、叶锈病、黄斑叶枯病）严重威胁小麦产量与品质，病害导致的产量损失可达10%-30%。传统人工检测方法依赖专家经验，存在耗时久、主观性强、田间覆盖范围有限等问题。

本文提出基于改进ResNet的双注意力机制模型（NGR），通过优化ResNet网络结构并融合全局注意力机制GAM和基于归一化的注意力模块NAM，解决小麦叶片病害“局部病灶特征模糊（如白粉病初期白色霉斑、叶锈病微小孢子堆）、相似病害区分难（如斑枯病与黄斑叶枯病症状相近）、复杂背景下特征提取效率低”的核心问题。实验结果表明，该模型在自建小麦叶片病害数据集上实现了98.78%的分类准确率，为小麦病害田间自动化诊断提供高效、可靠的技术方案。

2. 模型核心创新点

2.1 改进ResNet结构
优化ResNet50网络参数，调整池化策略，采用多尺度自适应窗口捕捉病斑边缘的细微纹理差异，同时减少下采样频率以保留叶片局部病变特征，提升模型计算效率和特征识别能力。

2.2 双注意力机制融合
GAM全局注意力机制：通过协同空间与通道注意力，精准定位病害区域并强化信息丰富的特征通道，实现多尺度目标检测。GAM包含通道注意力和空间注意力两个子模块，能够自适应地对不同的通道和空间位置进行加权，使网络更有效地学习和利用特征。
NAM归一化注意力模块：直接使用BN层的缩放因子自动计算注意力权重，无需额外全连接层和卷积操作，通过正则化项进一步抑制不显著特征，提升模型对关键特征的提取能力。

2.3 空间-语义协同感知
通过跨层残差连接融合浅层纹理信息与深层语义特征，建模病斑与健康组织间的空间关联及不同病害类型的语义关系，显著提升对白粉病、叶锈病、斑枯病等复杂病症的鉴别能力，增强模型在光照不均、叶片遮挡等复杂农田场景下的鲁棒性。

3. 实验数据集

3.1 数据集概况

本研究基于自建小麦叶片常见病害检测数据集（Wheat Leaf Disease Detection Dataset, WPLDD），数据集已随项目上传至仓库的?WPLDD/?文件夹，无需额外下载。
数据集名称	包含类别	图像总数	图像分辨率	数据分布（训练:测试）
WPLDD	白粉病、叶枯病、黄斑叶枯病、叶锈病、健康叶片	7400 张	统一 resize 至 224×224	3:1（通过代码自动划分）

3.2 数据集结构

仓库中?WPLDD/?文件夹组织如下，图像均采集于小麦主产区田间（涵盖不同生育期、光照与拍摄角度），可直接用于模型训练/测试：
WPLDD/
├── 叶枯病/       # 小麦枯萎病叶片图像（病斑特征：叶片枯萎、变色，严重时大面积干枯，影响植株生理功能）
├── 白粉病/       # 小麦白粉病叶片图像（病斑特征：叶片表面覆盖白色粉状霉层，阻碍气体交换与光合作用，导致叶片逐渐变黄、枯萎）
├── 黄斑叶枯病/   # 小麦斑枯病叶片图像（病斑特征：初期为褐色小斑点，随病情发展扩大并连接成片，对叶片损害较大）
├── 叶锈病/       # 小麦叶锈病叶片图像（病斑特征：产生锈色孢子堆，初期黄色或橙色，后期深褐色，使叶片失去光泽、功能衰退）
└── 健康植株/     # 健康小麦植株图像（特征：叶片鲜绿、形态舒展有光泽，整体生长旺盛，无病害或虫害迹象）
